# Contributing

Thanks for your interest in improving this guide. We welcome contributions from HVAC professionals, homeowners, and anyone with relevant experience.

## How to Contribute

1. Fork this repository
2. Create a branch for your changes
3. Make your edits
4. Submit a pull request with a clear description of what you changed and why

## What We're Looking For

- Corrections to technical information
- Additional troubleshooting scenarios
- Region-specific tips (especially South Florida)
- Improved clarity or organization
- New checklists or guides

## Guidelines

- Write for homeowners, not technicians — avoid jargon or explain it when used
- Include specific, actionable advice (not just "call a professional")
- Cite Florida Building Code or DBPR regulations when relevant
- Stick to the existing format and structure
