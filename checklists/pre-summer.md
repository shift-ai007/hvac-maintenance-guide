# Pre-Summer AC Checklist (March-April)

Complete these tasks before South Florida's summer heat arrives.

## Filter & Airflow
- [ ] Replace air filter (note size: _____)
- [ ] Clean all supply registers and return grilles
- [ ] Verify all vents are open and unobstructed
- [ ] Check for furniture or curtains blocking vents

## Outdoor Unit
- [ ] Clear all vegetation within 24 inches of condenser
- [ ] Remove leaves, dirt, and debris from unit
- [ ] Rinse condenser coils with garden hose (top to bottom)
- [ ] Verify condenser fan spins freely (power OFF first)
- [ ] Check unit is level on pad (use a level)
- [ ] Tighten mounting bolts if loose

## Drainage
- [ ] Flush condensate drain line with 1 cup white vinegar
- [ ] Inspect drain pan for cracks or rust
- [ ] Verify water flows freely from drain line outlet
- [ ] Test float switch (if equipped) by pouring water into pan

## Thermostat
- [ ] Replace batteries
- [ ] Verify temperature reading matches a separate thermometer
- [ ] Set to COOL mode, AUTO fan
- [ ] Program schedule if programmable/smart thermostat

## System Test
- [ ] Run system for 30 minutes
- [ ] Check airflow at each vent (should feel cool and strong)
- [ ] Listen for unusual noises (clicking, grinding, rattling)
- [ ] Check that indoor temp drops to within 15-20F of outdoor temp
- [ ] Inspect refrigerant line insulation for cracks or missing sections
