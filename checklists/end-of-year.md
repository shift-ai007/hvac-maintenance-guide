# End-of-Year HVAC Maintenance (November-December)

Year-end maintenance while the system runs less.

## Cleaning
- [ ] Clean condenser coils (final rinse of the year)
- [ ] Clean supply and return vents
- [ ] Flush condensate drain line with vinegar
- [ ] Wipe down drain pan
- [ ] Vacuum around indoor air handler

## Inspection
- [ ] Check ductwork for visible damage or disconnections
- [ ] Inspect refrigerant line insulation
- [ ] Check thermostat accuracy (compare to standalone thermometer)
- [ ] Test heat mode if applicable (heat pump)
- [ ] Listen for new/unusual sounds during operation

## Assessment
- [ ] System age: _____ years
- [ ] Number of repair calls this year: _____
- [ ] Total repair cost this year: $_____
- [ ] Current monthly electric bill vs. last year: $_____ vs $_____
- [ ] Any persistent comfort issues? _____

## Planning
- [ ] If system is 10+ years: get replacement quote for comparison
- [ ] If R-22 system: budget for replacement (R-22 phase-out in effect)
- [ ] Schedule annual tune-up for January/February (off-peak pricing)
- [ ] Stock up on air filters for the coming year
