# Maintenance Tools

Interactive tools to help South Florida homeowners optimize their HVAC maintenance.

## Available Tools

### savings-calculator.py

An interactive calculator that estimates how much you can save by maintaining your AC system properly. It considers:

- Filter replacement frequency
- Annual professional maintenance
- Outdoor coil cleanliness
- System age and efficiency
- Prevented emergency repairs
- Extended equipment life

```bash
python3 tools/savings-calculator.py
```

**Requirements:** Python 3.6+ (standard library only)
